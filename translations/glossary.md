## Rules

1. **Trainer** is always "Entrenador" (es) / "Treinador" (pt-br), never "Coach"
2. **Trainee** is always "Alumno" (es) / "Aluno" (pt-br), never "Cliente" or "Estudiante"
3. **Workout** is "Entrenamiento" (es) / "Treino" (pt-br), not "Ejercicio" / "Exercício" (that's Exercise)
4. **Macros** stays as "Macros" in all languages (universally understood)
5. **Set/Rep** are always the singular sport science terms, not generic translations
6. Capitalize consistently: title case for section headers, sentence case for descriptions
7. Use formal "you" (usted/você) only in formal contexts; default to informal (tú/você)
